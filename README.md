## Project Overview
This project analyzes retail transaction data to understand customer purchase behavior and revenue contribution patterns using statistical methods.

## Objectives
- Analyze order quantity as a random variable
- Compare UK vs Non-UK customer behavior
- Study revenue contribution patterns
- Apply probability and statistical reasoning

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- UK customers contribute a higher percentage of total revenue.
- Larger orders are more common in specific regions.
- Statistical testing suggests a relationship between customer location and order size.
